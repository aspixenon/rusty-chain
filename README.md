# Rusty Chain

A lightweight, Proof-of-Work blockchain implementation written in Rust.

## Features

- **SHA-256 Hashing**: Uses cryptographic hashing to link blocks.
- **Proof of Work**: Implements a mining algorithm with adjustable difficulty.
- **Validation**: Includes a chain integrity check mechanism.
- **Modular Architecture**: Clean separation of Block and Chain logic.

## Getting Started

### Prerequisites

Ensure you have Rust and Cargo installed.

### Installation

```bash
git clone [https://github.com/aspixenon/rusty-chain.git](https://github.com/aspixenon/rusty-chain.git)
cd rusty-chain
cargo build --release
